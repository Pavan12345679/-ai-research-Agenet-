# AI Response & Research Agent

An AI-powered agent that researches queries and generates accurate, context-aware responses, built to streamline information retrieval and summarization.

## Features
- Uses the OpenAI API to process and respond to research queries
- Generates context-aware, accurate responses
- Streamlines information retrieval and summarization for end users

## Tech Stack
- Python
- OpenAI API
- HTML
- CSS

## How to Run
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add your OpenAI API key to a `.env` file
4. Run the application: `python main.py`
